# MentorMind AI

MentorMind AI is an AI-powered mentorship platform that connects students with domain experts and provides intelligent guidance using AI.

## Features

* AI mentor chatbot
* Expert recommendation system
* Chat-based mentorship interface
* FastAPI backend
* React-based frontend

## Tech Stack

Frontend: React, JavaScript, CSS
Backend: FastAPI, Python
Database: SQLite
AI: Groq / LLM integration

## Project Structure

mentormind-ai
│
├── backend → FastAPI server
├── frontend → Web interface
└── README.md

## How to Run

### Backend

cd backend
pip install -r requirements.txt
uvicorn main:app --reload

### Frontend

cd frontend
npm install
npm start
